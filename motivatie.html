Hoe behoud ik mijn motivatie

Het behoud van motivatie kan lastig zijn, ik ben daarom van plan om niet te stoppen bij de opdrachten die wij van school krijgen.
Uit ervaring van mijn vorige studie ging ik snel door de stof heen en raakte ik daar door gedemotiveerd.
Ik ben van plan om ook de theorie te leren zodat ik mijn code beter kan beargumenteren.
Ook ben ik van plan trello te gebruiken om doelen te zetten, zodat ik zelf kan zien wat ik nog moet doen en wat ik af heb zo kan ik duidelijk zien hoe ik het groter doel kan bereiken in stappen.
